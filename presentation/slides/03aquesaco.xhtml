XForms quésaco ?

XForms est un langage de balisage XML pour une nouvelle génération d’applications sur le Web. XForms n’est pas principalement un type de document autonome, mais est destiné à être intégré à d’autres langages de balisage, tels que XHTML, ODF ou SVG.

Une application basée sur XForms collecte et traite des données en utilisant une architecture qui sépare la présentation, la finalité et le contenu ; le traitement des données s’effectue via trois mécanismes :

  • un modèle déclaratif composé de formules pour les calculs et contraintes, de déclarations de types de données et d’autres propriétés, ainsi que de paramètres pour la soumission des données ;
  • une couche vue composée de contrôles d’interface utilisateur basés sur l’intention ;
  • un contrôleur impératif pour orchestrer la manipulation des données, les interactions entre les couches modèle et vue, et la soumission des données.

Ainsi, XForms permet la réutilisation de composants, favorise une validation stricte des types de données, élimine les aller-retours inutiles avec le serveur, offre l’indépendance vis-à-vis des dispositifs et l’accessibilité, et réduit le besoin de scripts.
