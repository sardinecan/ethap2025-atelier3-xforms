Contrôleur - Validation

XForms propose bien évidemment des fonctionnalités avancées pour contraindre les données :

Ces contraintes doivent être déclarées dans le <xf:model/> à l’aide de la balise <xf:bind/>

  • champs requis : @required
  • types XML : @type
  • contrainte : @constraint
  • valeur calculée : @calculate
