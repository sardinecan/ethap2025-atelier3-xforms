Contrôleur - Soumission

XForms propose un module de soumission très complet comprenant de multiples paramètres de sérialisation et compatible avec de nombreux protocoles. Ces paramètres sont à déclarer dans le <xf:model> avec la balise <xf:submission/>.

  • Méthode de soumission : @method
  • Type MIME : @mediatype
  • Destination : @resource
