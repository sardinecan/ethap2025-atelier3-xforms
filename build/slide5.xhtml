Fonctionnement de XForms

Séparer les valeurs des contrôles

Le modèle spécifie les valeurs collectées (l’instance) et leur logique

  • types, restrictions
  • valeurs initiales, relations entre les valeurs

Le corps du document lie (bind) des contrôles de formulaires aux valeurs dans l’instance

Des contrôles fondés sur l’intention

Les contrôles de formulaires expriment leur fonction (sélectionner une valeur dans une liste, etc.) plutôt que ce à quoi les champs doivent ressembler (bouton radios, menus, etc.).

Il est ainsi possible d’indiquer comment ils doivent être représentés sur différents terminaux.
